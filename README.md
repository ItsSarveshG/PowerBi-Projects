**Project Overview**

The **Data Professional Survey Dashboard** offers an interactive and visual representation of:
1. Distribution of job titles across the data field
2. Average salaries based on roles and experience levels
3. Popular programming languages and tools used by professionals
4. Work satisfaction and career growth insights
5. Demographic breakdowns of survey respondents


**Key Insights**

1. Identifies trends in the data industry
2. Compares salaries by country, experience, and role
3. Highlights the most commonly used tools and technologies
4. Visualizes overall job satisfaction among data professionals


**How to Use**

Simply open the _**.pbix**_ file in Power BI Desktop to explore the dashboard interactively.


**Tools Used**

1. Power BI Desktop for data modeling and visualization
2. Excel/CSV as the primary data source (survey dataset)
